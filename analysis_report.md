# Sales Data Analysis Report

## Project Overview
This project analyzes real-world sales data using Python Pandas.

## Objectives
- Load and explore sales data
- Clean missing values
- Calculate key sales metrics

## Dataset
Rows: 100  
Columns: Date, Product, Quantity, Price, Customer_ID, Region, Total_Sales

## Metrics Calculated
- Total Revenue
- Average Sales
- Highest and Lowest Sale
- Best Product
- Best Region

## Conclusion
The analysis shows strong performance from high-value products and specific regions.
